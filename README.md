<h1>Open32</h1>
<p><i>A tiny Esp32 based console.</i></p>
<p><b>1.</b> Build the project as shown in the schematics.</p>
<p><b>2.</b> Download bootloader.ino and flash it on the Esp32.</p>
<p><b>3.</b> Connect the device to the internet and update it.</p>
<p><b>4.</b> Choose a game and you're ready to play.</p>
<p><b>5.</b> Create your own games for the community.</p>
